# 🍕 Python Pizza Delivery

A beginner-friendly Python program that calculates the total cost of a pizza order based on the selected pizza size and optional toppings.

---

## ✨ Features

- Choose from three pizza sizes: **Small**, **Medium**, or **Large**.
- Add pepperoni as an optional topping.
- Add extra cheese.
- Automatically calculates the final bill.
- Uses user input to customize the order.

---

## 📚 Concepts Practiced

- Variables
- User Input
- String Methods (`.upper()`)
- Conditional Statements (`if`, `elif`, `else`)
- Nested Conditions
- Arithmetic Operations
- Formatted Strings (f-strings)

---

## 🚀 How to Run

1. Make sure **Python 3** is installed.
2. Clone this repository or download the source code.
3. Open a terminal in the project folder.
4. Run the program:

```bash
python pizza_delivery.py
```

## ⚠️ Current Limitations

- Assumes the user enters valid input.
- Invalid pizza sizes or topping choices are not validated.
- Processes only one pizza order per execution.
- Menu items and prices are hardcoded in the source code.
